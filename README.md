# Prompt Optimizer

将杂乱的想法转化为结构清晰、逻辑严密的 Prompt，用于辅助 AI 编程（Vibe Coding）。

## 功能特性

- 💬 聊天式交互界面，用户输入在右，AI 优化结果在左
- ✨ 一键优化，AI 智能生成结构化 Prompt
- 📋 一键复制生成的 Prompt 到剪贴板
- 💾 对话历史自动保存（localStorage）
- 📱 响应式布局，支持移动端

## 快速开始

npm install
npm run dev

## 部署到 Vercel

1. Fork 或导入此仓库到 GitHub
2. 在 Vercel 中导入项目
3. 添加环境变量：
   - `AGNES_API_KEY`：你的 Agnes API Key
4. 部署完成后访问

## 技术栈

- React 19
- Vite
- Tailwind CSS 4
- react-markdown
- Vercel Serverless Functions

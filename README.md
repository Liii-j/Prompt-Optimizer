# Prompt Optimizer

将杂乱的想法转化为结构清晰、逻辑严密的 Prompt，用于辅助 AI 编程（Vibe Coding）。

## 功能特性

- 💬 聊天式交互界面，用户输入在右，AI 优化结果在左
- ✨ 一键优化，本地规则处理（预留 LLM API 接口）
- 📋 一键复制生成的 Prompt 到剪贴板
- 💾 对话历史自动保存（localStorage）
- 📱 响应式布局，支持移动端

## 快速开始

npm install
npm run dev

## 构建部署

npm run build
npm run preview

## 技术栈

- React 19
- Vite
- Tailwind CSS 4
- react-markdown
